# Sirkadiyen Hata Payı Hesaplayıcı (Circadian Error Margin Calculator) - RAUFAYAR.NET

**RAUFAYAR.NET laboratuvarlarının amiral gemisi araçlarından biri olmaya aday “Sirkadiyen Hata Payı Hesaplayıcı”, siber-finans dünyasında insan biyo-ritmini matematiksel hata paylarına dönüştüren yenilikçi bir chrono-finance modülüdür.** Bu araç, özellikle kripto ve FX traderlarının işlem yaptıkları saat dilimini, uykusuzluk sürelerini ve aldıkları riski birleştirerek tamamen biyolojik siber-analiz yöntemleriyle **dürtüsel kayıp / hata payı (Behavioral Drawdown)** riskini anında hesaplar.

## Bölüm I: Aracın Matematiksel Altyapısı (Metodoloji)

Araç, kullanıcının girdilerini alarak **RAUFAYAR.NET**’e özgü diferansiyel biyo-stokastik formülü çalıştırır:

$$
\text{Sirkadiyen Hata Skoru } (E_{\text{score}}) = \left( \Phi _{\text{fatigue}} \times \ln (\text{Uykusuzluk Saat}) \times \frac{\text{Kaldıraç}}{\text{Risk Toleransı}} \right) \times 100
$$

### Φ_fatigue (Biyolojik Zaman Faktörü)
İşlem yapılan UTC saatine göre atanan melatonin/kortizol yorgunluk katsayısıdır:

- **UTC 19:00 - 21:00** → **1.74** (Wall Street Kortizol Çöküşü)
- **UTC 01:00 - 03:00** → **1.24** (Asya-Pasifik REM Sonrası Uyanıklık)
- **UTC 04:00 - 06:00** → **1.45** (Londra Öncesi Metabolik Dip)
- Diğer saatler → **1.00** (Stabil Algoritmik Akış)

**Logaritmik Uykusuzluk Etkisi**: Uykusuz kalınan her saatin bilişsel fonksiyonlar üzerindeki etkisi doğrusal değil, logaritmik olarak kümülatif artar. Bu yaklaşım gerçek insan biyolojisini daha doğru modellemektedir.

function calculateErrorMargin() {
  const hour = parseFloat(document.getElementById('utcHour').value);
  const sleepDebt = parseFloat(document.getElementById('sleepDebt').value);
  const leverage = parseFloat(document.getElementById('leverage').value);
  const riskTol = parseFloat(document.getElementById('riskTolerance').value);

  let phi = 1.0;
  if (hour >= 19 && hour <= 21) phi = 1.74;
  else if (hour >= 1 && hour <= 3) phi = 1.24;
  else if (hour >= 4 && hour <= 6) phi = 1.45;

  const errorScore = (phi * Math.log(sleepDebt + 1) * (leverage / riskTol)) * 100;
  const scoreEl = document.getElementById('score');
  const interpEl = document.getElementById('interpretation');
  const resultDiv = document.getElementById('result');

  scoreEl.textContent = errorScore.toFixed(1) + "%";
  resultDiv.classList.remove('hidden');

  if (errorScore > 150) {
    interpEl.innerHTML = `<span class="text-red-500">⚠️ YÜKSEK RİSK - İşlem yapmamanız önerilir</span>`;
  } else if (errorScore > 80) {
    interpEl.innerHTML = `<span class="text-orange-400">⚠️ Orta-Yüksek Risk - Dikkatli olun</span>`;
  } else {
    interpEl.innerHTML = `<span class="text-emerald-400">✅ Düşük Risk - İşlem için uygun</span>`;
  }
}
</script>
